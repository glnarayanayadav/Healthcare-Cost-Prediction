# Healthcare Cost Prediction

End-to-end Machine Learning project for predicting treatment costs.

## Tasks
- Regression: Predict continuous estimated_treatment_cost
- Classification: Predict cost tier (0/1)

## Project Structure
- data/ : Raw datasets
- src/ : Source code
- models/ : Saved models
- outputs/ : Prediction outputs
- notebooks/ : Jupyter notebooks

## Run

Regression:
python src/train_regression.py

Classification:
python src/train_classification.py
